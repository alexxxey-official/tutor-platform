Теория

Что такое квадратные уравнения?

Стандартный вид

Квадратное уравнение — это уравнение, старшая степень неизвестной в котором равна 2. Его общий вид:

\[ax^2 + bx + c = 0\]

Где \(a\neq 0\), а коэффициенты \(a, b, c\) — любые действительные числа.

Формула Дискриминанта

Универсальный способ найти корни — посчитать дискриминант \(D\).

\[D = b^2 - 4ac\] \[x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}\]
⚠️ Сколько корней?
Если \(D > 0\) — 2 корня.
Если \(D = 0\) — 1 корень (или 2 совпадающих).
Если \(D < 0\) — действительных корней нет.

Теорема Виета

Для приведённого уравнения (где \(a = 1\)), то есть \(x^2 + px + q = 0\), справедливы формулы:

\[x_1 + x_2 = -p \qquad \text{и} \qquad x_1 \cdot x_2 = q\]

Это позволяет находить корни устно, подбирая множители числа \(q\), дающие в сумме \(-p\).

Разборы

Решённые примеры

Пример 1 · Через дискриминантШаг за шагом
Решить уравнение: \(2x^2 - 5x + 3 = 0\)
1
Выпишем коэффициенты
\(a = 2,\; b = -5,\; c = 3\)
2
Вычислим дискриминант \(D = b^2 - 4ac\)
\(D = (-5)^2 - 4 \cdot 2 \cdot 3 = 25 - 24 = 1\)
3
Найдём корни по формуле \(x = \frac{-b \pm \sqrt{D}}{2a}\)
\(x_1 = \frac{5 + 1}{4} = \frac{6}{4} = 1.5\)
\(x_2 = \frac{5 - 1}{4} = \frac{4}{4} = 1\)
Ответ\(\; x_1 = 1.5,\; x_2 = 1\)
Пример 2 · Неполное квадратное (c=0)Шаг за шагом
Решить уравнение: \(x^2 - 7x = 0\)
1
Вынесем общий множитель \(x\) за скобку
\(x(x - 7) = 0\)
2
Произведение равно нулю, когда хотя бы один из множителей равен нулю
\(x = 0 \quad \text{или} \quad x - 7 = 0\)
Ответ\(\; x_1 = 0,\; x_2 = 7\)
Задания

Упражнения для занятия

ℹ️ Вводи корни через запятую без пробелов (например: 2,-5). Если корень один, введи только его (например: 3). Порядок ввода корней не важен.

Уровень 1 — Неполные и простые

Без дискриминанта
1.
Решить уравнение: \(x^2 - 9 = 0\)
Перенеси 9 вправо: \(x^2 = 9\). Какое число в квадрате даёт 9? Не забудь про минус!
2.
Решить уравнение: \(x^2 + 5x = 0\)
Вынеси \(x\) за скобку: \(x(x+5)=0\). Приравняй каждый множитель к нулю.

Уровень 2 — Полные уравнения

Дискриминант или Виет
3.
Решить уравнение: \(x^2 - 7x + 10 = 0\)
По теореме Виета: сумма равна \(7\), произведение \(10\). Какие это два числа?
4.
Решить уравнение: \(x^2 + 6x + 9 = 0\)
Здесь \(D=0\) или используй формулу квадрата суммы: \((x+3)^2 = 0\).

Уровень 3 — Более сложные

Коэффициент \(a \neq 1\)
5.
Решить уравнение: \(2x^2 - 3x - 5 = 0\)
\(D = (-3)^2 - 4(2)(-5) = 9 + 40 = 49\). Корень из \(D\) равен \(7\).
6.
★ Привести к квадратному и решить: \(x + \dfrac{4}{x} = 5\)
Умножь всё уравнение на \(x\) (при условии \(x \neq 0\)). Получишь \(x^2 - 5x + 4 = 0\).
Прогресс занятия 0 / 6 правильно
Справочник

📋 Шпаргалка — Квадратные уравнения

Дискриминант

\(D = b^2 - 4ac\)

\(x_{1,2} = \dfrac{-b \pm \sqrt{D}}{2a}\)

Теорема Виета

Для \(x^2 + px + q = 0\):
\(x_1 + x_2 = -p\)
\(x_1 \cdot x_2 = q\)

Неполное: \(c = 0\)

\(ax^2 + bx = 0\)
\(x(ax + b) = 0\)
Корни: \(x=0\) и \(x = -b/a\)

Неполное: \(b = 0\)

\(ax^2 + c = 0\)
\(ax^2 = -c\)
Извлекаем корень: \(x = \pm \sqrt{-c/a}\)
