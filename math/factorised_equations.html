Теория

Zero Product Property

Свойство нулевого произведения

Если произведение нескольких множителей равно нулю, то хотя бы один из них обязан быть равен нулю. Это главный принцип решения уравнений разложенных на множители (factorised equations).

\[A \cdot B = 0 \iff A = 0 \quad \text{или} \quad B = 0\]

Уравнения высших степеней

Чтобы решить уравнение степени 2 и выше, часто нужно сперва перенести все слагаемые в одну сторону (чтобы справа был \(0\)), а затем разложить левую часть на множители (factorise).

\[x^2 - 5x = 0 \implies x(x - 5) = 0 \implies x = 0 \text{ или } x = 5\]
⚠️ Частая ошибка (Warning!) Никогда не дели обе части уравнения на \(x\)! Так ты "потеряешь" корень \(x=0\). Всегда выноси \(x\) за скобки.
Разборы

Решённые примеры

Пример 1 · Линейные скобкиШаг за шагом
Solve: \((x - 3)(2x + 8) = 0\)
1
Приравниваем первый множитель к нулю
\(x - 3 = 0 \implies x = 3\)
2
Приравниваем второй множитель к нулю
\(2x + 8 = 0 \implies 2x = -8 \implies x = -4\)
Ответ\(\;x_1 = 3,\; x_2 = -4\)
Пример 2 · Комбинированное уравнениеШаг за шагом
Solve: \(x(x^2 - 25) = 0\)
1
Отдельный множитель \(x\) даёт первый корень
\(x = 0\)
2
Вторая скобка — разность квадратов (difference of squares)
\(x^2 - 25 = 0 \implies x^2 = 25 \implies x = \pm 5\)
Ответ\(\;x = 0,\; x = 5,\; x = -5\)

Уровень 1 — Базовый

Два простых множителя

Solve: \((x - 2)(x - 5) = 0\)
2.
Solve: \(x(x + 3) = 0\)
Сам \(x\) перед скобкой — это тоже множитель. Значит \(x=0\) — один из корней.
3.
Solve: \((2x - 4)(x + 1) = 0\)
4.
Solve: \((x + 7)(x - 7) = 0\)

Уровень 2 — Средний

Квадратные выражения и три множителя
5.
Solve: \((x^2 - 4)(x - 1) = 0\)
Уравнение \(x^2 - 4 = 0\) имеет два корня: \(2\) и \(-2\). Итого будет 3 корня.
6.
Solve: \(x(x^2 - 9) = 0\)
7.
Solve: \((3x - 6)(2x + 8) = 0\)
8.
Solve: \(x^2(x - 5) = 0\)
\(x^2=0\) означает просто \(x=0\) (это называется кратный корень, но в ответ пишем один раз).

Уровень 3 — Сложный

Неполные уравнения высших степеней
9.
Solve: \((x^2 - 25)(x^2 - 1) = 0\)
10.
Solve: \(x^3 - x = 0\)
Сначала вынеси \(x\) за скобки: \(x(x^2 - 1) = 0\).
11.
Solve: \((x^2 + 4)(x - 3) = 0\)
Может ли \(x^2 + 4\) равняться нулю? \(x^2 = -4\) не имеет действительных корней! Поэтому корень только из второй скобки.
12.
★ Solve: \(x^4 - 16 = 0\)
Разложи как разность квадратов: \((x^2 - 4)(x^2 + 4) = 0\).
Прогресс классной работы 0 / 12 правильно
Справочник

📋 Шпаргалка — Factorised Equations

Главное правило

\(A \cdot B \cdot C = 0\)

Значит \(A=0\), или \(B=0\), или \(C=0\).

Common factor (Общий множитель)

\(ax^2 + bx = 0\)
\(x(ax + b) = 0\)
Корни: \(x=0\) и \(x=-b/a\).

Difference of squares

\(x^2 - a^2 = 0\)
\((x-a)(x+a) = 0\)
Пример: \(x^2 - 36 = 0 \implies x = \pm 6\)

No real roots (Нет корней)

\(x^2 + a = 0\) (где \(a > 0\))

Пример: \(x^2 + 9 = 0\) не даёт решений (в действительных числах).
